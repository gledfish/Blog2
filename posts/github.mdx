## github的基本使用
### 创建仓库
如下图所示，单击Create a new repository 可以创建你自己的仓库。
![image.png](https://s2.loli.net/2022/12/11/EnXKlGLqCspDacY.png)
创建仓库有以下几个主要步骤
1. 输入仓库名
2. 在Description中你可以选择描述以下你的仓库
3. 然后选择仓库是否公开（如果选择public,意味着别人也能看到你的仓库内容。而private则是由你自己做主。）
4. 单击 Create repository
创建成功的画面
![image.png](https://s2.loli.net/2022/12/11/6cD5KZu9PASyrok.png)
这样你自己的项目仓库就建好了。那如何往里面添加代码文件呢？
### 添加代码文件
添加代码文件有多种方法，这里讲解最简单的方法：使用Github desktop。

首先你需要下载这个软件。
![image.png](https://s2.loli.net/2022/12/11/6cD5KZu9PASyrok.png)
单击右上角绿色的Code，选择Open with Github Desktop。等待下载完成安装完成后，回到我们的仓库页面。（什么？你找不到你的仓库了？先回到官网，然后单击右上角你的头像，单击repositories。）

单击branch,然后点击右上角绿色的 New branch，选择分支名。（创建一个分支，你可以在分支上肆无忌惮的操作你的代码，而不会影响主干中的代码，同时，如果主干中的代码出现问题，也可以根据分支来恢复
![image.png](https://s2.loli.net/2022/12/11/M8o9bQAN5w3gzEG.png)
