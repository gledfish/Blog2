## 遗漏
### global关键字
块结构中定义的变量，往往只有块作用域，如果希望定义全局变量，使用global
```python
def myFunc():
    x = 5 # 块作用域
    global y = 5 #全局作用域
```

### 多行字符串赋值
多行的字符串赋值
```python
a = """This is a long long long 
long long long long string."""
print(a)
```

### 逆序截取字符串
```python
b = "Hello, world!"
b[-5:-2]# orl
```

### 移除字符串中的空白格
```
a = " Hello, world! "
b = a.strip()# "Hello,world!"
```

### 逻辑运算符
```python
x, y = 1, 2
x and y;#True
x or y;# True
x not y; #True
```
### 对象识别运算符
```python
x is y #如果x 和 y 是同一个对象，返回True， 否则返回False
x is not y
```

### 成员运算符
```Python
1 in [1, 3, 5] #True
1 not in [1, 3, 5] #False
```

### Python Virtual Environments
为每个项目创建一个独立的运行环境
1. 创建一个新的目录
2. 在该目录中使用命令行指令：python -m venv project_name
3. 在该目录下找到activate.bat文件并运行
4. 现在你已经有了一个独立的运行环境，再创建一个requirements.txt文件记录项目依赖。
