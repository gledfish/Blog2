# Python Virtual Environments
为每个项目创建一个独立的运行环境
1. 创建一个新的目录
2. 在该目录中使用命令行指令：python -m venv project_name
![image.png](https://s2.loli.net/2022/12/16/dnLoZFXTPjVJrNS.png)
3. 在该目录下找到 activate.bat文件并运行
![image.png](https://s2.loli.net/2022/12/16/SITbGUh1vx9YNyP.png)
4. 现在你已经有了一个独立的运行环境，这是当前环境中的依赖。
