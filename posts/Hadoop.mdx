# Hadoop教程
## 简介
由 Apache 基金会发布的分布式系统基础架构

**优点**
* 高可靠性
* 高拓展性
* 高效性
* 高容错性

**HDFS架构**
Hadoop分布式文件系统架构
NameNode：存储 DataNode 存储文件的信息
DataNode：存储数据
**Yarn架构**
RM：resource manager 集群资源中心
NM：Node manager 单结点服务器资源中心
Am：application Master 单个任务的中心
**MapReduce**
Map 阶段：并行处理输入数据
Reduce 阶段：对Map阶段的结果进行汇总
