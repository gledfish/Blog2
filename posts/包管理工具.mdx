# Windows包管理工具对比
Windows 常用的包管理工具主要有以下：**Scoop，
Chocolatey，npm, yum，apt**
本文将主要通过以下指标来衡量
* 安装教程是否详细
* 操作是否方便
* 包内容是否全面

## npm
### 安装和删除
npm是js中的包管理工具，并且是Node.js的默认包管理工具。安装Node.js时会附带安装
### 操作列表
![image.png](https://s2.loli.net/2022/12/11/nso8vYGS3RKCc71.png)
### 具体操作
'''
npm help command
'''
会跳转到官方的文档网页，进行详细介绍。如：npm help install
![image.png](https://s2.loli.net/2022/12/11/gQxwqLomC3yFIGV.png)
### 包内容
包内容达两百万之多，但是内容只局限于编程领域。
### 总结
npm包是编程时不可缺少的利器，但是对于一般用户作用就不是很大了。
## apt
apt是ubuntu常用的包管理系统之一，（Ubuntu是Linux系统的一种发行版）同时也是ubuntu系统自带的包管理系统。
### 操作列表
![image.png](https://s2.loli.net/2022/12/11/E9KAolZ3i8smPaN.png)
### 包内容
包内容的多少取决于软件源的包数量。推荐使用清华源或者阿里源。
## 总结
这些包管理工具的操作往往极其相似。
Scoop官网页面简洁，教程醒目，简洁明了。更适合一般用户。
而程序员则需要根据项目的需要使用对应的包管理工具。
