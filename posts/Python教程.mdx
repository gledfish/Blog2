# Python的程序结构
一个Python程序的代码往往是以下几个部分：
1. 引入外部包
2. 定义函数和类
3. 对这些包，函数，类以及基本的数据类型进行运算
4. 检验运算结果
也许上面的某些术语你并不完全理解，但是不用担心，你可以放心的阅读下面的部分。
## Python变量及数据类型
Python的数据类型主要有Python自身预定义的数据类型，和用户定义的数据类型。Python自身定义的数据类型，主要包括一些数字类型，如整数（int），浮点数（float,又叫小数），和不太常用的复数类型（complex）。文本类型，如字符串（str）,判断正误的bool类型。

Python中的变量就像数学中方程的解，只有当你解出这个方程时，你才会知道这个x，是一个整数还是一个小数，亦或者像汉字中的灯谜。你没有猜出来时，你并不知道它的信息。Python中变量不需要声明，也不需要预先确定它的数据类型，当它被赋予特定的值时，它的本身被创建，数据类型也就确定了。

有了变量和数据类型的概念，便可以写一些简单的Python语句

```python
x = 5# 这是一条注释，不参与程序的运行。 
y = 'John'#这里的 '='号不同于数学中表示相等，这里更应该从他的形状上看，它就像是一条管道，从上面看的样子，而=后面的数据从管道中传递给了y。（更多有关运算符的知识在后文会讲到。）这时y的数据类型确定了是一个字符串（str）类型
# y = "john" 和上面那条语句是等价的。
z = 5.2 #
```

你也可以在赋值时，就自主确定它的运算符
```python
x = int(3) # int
y = float(3) # float
z = str(3) # string
```
### 变量的命名
变量的命名首先要具有可读性（你也不想别人看到你的代码不知所云吧），而对于一些使用次数很少的变量的命名要简介，此外，变量的命名还要遵循一定的标准和规矩，总结一下，变量的命名规则如下：

1. 由字母或者下划线开头
2. 由字母，数字，下划线组成
3. 注意大小写（age和Age是不同的变量）
