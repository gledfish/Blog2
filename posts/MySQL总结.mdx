# MySQl
## 部署
打开 MySQL 的命令行工具
* mysql -u root -p
* 输入密码
## 日常增删查改
```sql
show databases; -- 展示当前所有的数据库
USE dbname; --选择要操作的数据库
SHOW TABLES; --看下数据库的大致内容，选择要操作的表
--不清楚表的具体内容？用下面这个
DESCRIBE tablename;
```
### 增
```sql
INSERT INTO tablename(c1,c2,...)
VALUES (v1,v2,...);--也可以一次性添加多行，用 , 号隔开

UPDATE tablename SET content WHERE condition;--在表中满足条件的地方设置值
```

### 删
```sql
DELETE FROM tablename WHERE condition;--删除复合特定条件的值
DELETE FROM tablename;--删除整个表
```
### 查
```sql
SELECT 
    select_list--筛选内容
FROM
    tablename--表名
INNER JOIN tablename ON expression --如果可能还需要别的表
WHERE
    search_condition --初步筛选条件
GROUP BY 
    expression -- 如果有必要分组
Having 
    condition --检验成果
```
```sql 
REPLACE INTO tablename(col1,col2)
VALUES(val1, val2);--一般第一个参数用于确定行位置
```
## 表管理
```sql
CREATE TABLE tablename(
    columnname datatype data-config,
) tabletype;-- 创建表

RENAME TABLE old_table_name TO new_table_name;--重命名

ALTER TABLE table_name DROP column_name;--删除列

ALTER TABLE table ADD [COLUMN] column_name column_definition [FIRST|AFTER existing_colum_name];--在给定列之前或之后创建新列。

DROP TABLE tablename;--删除列
```
