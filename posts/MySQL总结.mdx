# MySQl
## 部署
打开MySQL的命令行工具
* mysql -u root -p
* 输入密码
## 日常增删查改
```sql
show databases; -- 展示当前所有的数据库
USE dbname; --选择要操作的数据库
SHOW TABLES; --看下数据库的大致内容，选择要操作的表
--不清楚表的具体内容？用下面这个
DESCRIBE tablename;
```
### 增
```sql
INSERT INTO tablename(c1,c2,...)
VALUES (v1,v2,...);--也可以一次性添加多行，用,号隔开

```
### 查
```sql
SELECT 
    select_list--筛选内容
FROM
    tablename--表名
INNER JOIN tablename ON expression --如果可能还需要别的表
WHERE
    search_condition --初步筛选条件
GROUP BY 
    expression -- 如果有必要分组
Having 
    condition --检验成果

```
